## 0x06. Regular expression

In this project, I learn all about Regular Expressions, and how to use them in searching patterns. I learned the following concepts:
- The basics of Regular Expressions
- Using `grep` in Regular Expressions
- - Metacharacters
- Character Ranges
- Repititions
- Optional Characters
- Whitespace
- Wild characters
- Matching groups of characters, words, patterns
- Using <strong><a href="https://rubular.com/">Rubular</a></strong>
- Ruby on the Bash command line
